> [!note]
> Это выжимка для работы в Obsidian. Полный документ (с таблицами, DDL, моделью БД) — [открыть оригинал на диске](file:///Users/lavrov/Documents/Claude/Projects/FK.market/FK_market_TZ.md).

## Зачем

**FK.market** — онлайн-магазин цифровых товаров на платежах FKwallet. MVP v1 — платформа сама продаёт из своего каталога. v2 — мультивендорный маркетплейс (пользователи добавляют свои товары).

## Для кого

- Гость — просмотр и покупка без регистрации
- [[Глоссарий#`customer`|Покупатель]] — ЛК, история заказов, повторная выдача, баланс
- [[Глоссарий#`admin`|Администратор]] — админка
- [[Глоссарий#`seller`|Продавец]] — кабинет продавца (v2)

## Типы товаров

- **Цифровые ключи/коды** — Steam, VPN, лицензии ПО
- **Аккаунты** — стриминговые сервисы, VPN
- **ИИ-аккаунты** — ChatGPT Plus, Claude Pro, Midjourney, OpenAI API
- **Скины Steam** — CS2, Dota2
- **Подписки и пополнения** — Spotify, Steam
- **Услуги** — накрутки, буст

## Способы получения (`fulfillment_mode`)

См. [[Глоссарий#`fulfillment_mode`|термин]].

- **`stock`** — ключ из [[Глоссарий#`stock_items`|склада]], мгновенно
- **`api`** — запрос к API [[Глоссарий#`provider`|поставщика]] в момент оплаты
- **`manual`** — задача админу в очередь выдачи

С fallback'ом: `api → manual` при ошибке поставщика.

## Функционал

### Витрина
- Главная (баннер, категории, топ товаров)
- Каталог (фильтры, сортировки, пагинация)
- Карточка товара (галерея, описание, рейтинг, отзывы, покупка)
- Поиск (FULLTEXT)

### Оформление
- Корзина без регистрации
- Email обязателен (туда идёт выдача)
- Ввод параметров для api/manual товаров (логин Steam, Trade URL)
- Выбор способа оплаты (карта, СБП, кошелёк, крипта)

### Личный кабинет
- История заказов
- Повторная выдача купленных кодов
- Внутренний баланс (пополнение через FKwallet)
- Избранное, отзывы

### Админка
- CRUD товаров с `fulfillment_mode`
- Массовая загрузка ключей
- Управление категориями, наценками, промокодами
- Заказы со статусами
- [[Глоссарий#`fulfillment_tasks`|Очередь ручной выдачи]] с SLA
- [[Глоссарий#`ProviderGateway`|Поставщики API]] (статус, каталог, логи)
- Статистика (выручка, заказы, средний чек, маржа, топ товаров)
- Аудит-лог

## Интеграции

- **FKwallet** — приём платежей + выплаты (v2)
- **Digiseller / Kinguin / G2G** — API-поставщики
- **Steam-бот** — скины через трейды
- **OAuth** — VK ID, Яндекс ID, Telegram Login Widget
- **Email** — SMTP или UniSender / Mailgun

## Риски

Зафиксированы в [[Журнал решений]]:
- Блокировка платёжного провайдера → второй провайдер через `PaymentGateway`
- Фрод и возвраты → в оферте «возврат цифрового товара невозможен»
- ToS ИИ-сервисов → гарантия работоспособности N дней, бюджет на замены
- Steam Guard (скины) → 15-дневное ожидание перед трейдами
