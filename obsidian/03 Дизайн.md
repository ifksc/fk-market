# Дизайн

## Концепция

Современный гибрид с переключателем **light / dark**. Аудитория — игровая + общий рынок цифровых товаров, поэтому важно работать в обеих темах.

- Палитра: indigo → violet → pink (брендовый градиент)
- Акцент: emerald (успех, «в наличии»)
- Шрифт: Inter (400, 500, 600, 700, 800)

## Дизайн-система

- Общая шапка/футер — в `design/assets/layout.js` (подставляются по data-атрибутам).
- Переключатель темы — в `design/assets/theme.js` (сохраняется в `localStorage`).
- Кастомные стили — `design/assets/app.css` (градиент логотипа, тени, анимации).
- Сетка карточек: 2 / 3 / 4 колонки (mobile / tablet / desktop).
- Радиус: `rounded-xl` (12px) и `rounded-2xl` (16px).

## Что учли в UI

- **Режимы выдачи** — на админке показаны бейджем: Склад (indigo) / API (blue) / Ручная (purple) / API↓Ручная (fallback).
- **ИИ-аккаунты** — бейдж `NEW` в меню и на плитке категории.
- **OAuth VK / Яндекс / Telegram** — цветные кнопки на экране входа.
- **Очередь ручной выдачи** — карточки с SLA-таймером, просроченные подсвечиваются красным.
- **Поставщики API** — тумблеры включения, статусы «OK / Задержки / Выключен».

## Миграция в Next.js

Разметка Tailwind 1:1 переносится в React-компоненты. При переносе:
1. Шапка → `components/Header.tsx`
2. Футер → `components/Footer.tsx`
3. Карточка товара → `components/ProductCard.tsx`
4. Кнопки/инпуты → `components/ui/Button.tsx` и т. д.
5. Тема — через next-themes (работает как `localStorage`, но SSR-safe)
