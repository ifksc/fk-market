Главный путь заказа

orders → order_items → для каждого item в зависимости от products.fulfillment_mode:
• stock — резервируется stock_items и копируется в delivered_payload;
• api — создаётся fulfillment_tasks с режимом api, джоба вызывает ProviderGateway;
• manual — создаётся fulfillment_tasks с режимом manual, появляется в админ-очереди.

Шифрование

На уровне приложения (Laravel Crypt) шифруются: stock_items.payload, order_items.delivered_payload, providers.credentials. Ключ — в APP_KEY в .env.

Расчёт цены

products.price_final денормализован (чтоб не считать на каждом запросе). Пересчёт — при изменении price_base, markup_pct или pricing_rules. Правила применяются по убыванию priority.
