# FK.market — HTML-прототипы

Кликабельные макеты витрины и админки. Работают в любом браузере без сервера.

## Как смотреть

Откройте `index.html` двойным кликом — остальные страницы доступны через шапку, карточки товаров, корзину и подвал.

## Экраны

| Файл | Экран |
|------|-------|
| `index.html` | Главная — hero, категории, топ товаров, CTA «стать продавцом» |
| `catalog.html` | Каталог с фильтрами (цена, тип, рейтинг, способ выдачи) и пагинацией |
| `product.html` | Карточка товара — галерея, описание, отзывы, панель покупки |
| `cart.html` | Корзина — товары, промокод, итог, степпер |
| `checkout.html` | Оформление — контакты, параметры товаров, выбор платежа FKwallet, превью «Заказ оплачен» |
| `account.html` | Личный кабинет — баланс, заказы, повторная выдача, история транзакций |
| `login.html` | Вход — VK, Яндекс, Telegram + email/пароль |
| `admin.html` | Админка — KPI, график выручки, заказы, товары, склад, наценки |
| `logo.html` | Бренд-бук — 4 концепции логотипа, палитра, типографика |

## Общие файлы

| Файл | Назначение |
|------|-----------|
| `assets/app.css` | Кастомные стили: градиент логотипа, тени, анимации |
| `assets/theme.js` | Переключение темы light/dark, сохранение в localStorage |
| `assets/layout.js` | Общий header/footer — подставляется на все страницы кроме `login.html` и `admin.html` |
| `assets/logo-a.svg` | Текущий логотип-знак (Squircle) |
| `assets/logo-b.svg` | Концепт B — Shield & Bolt |
| `assets/logo-c.svg` | Концепт C — Ring |
| `assets/logo-d.svg` | Концепт D — Monogram |
| `assets/logo-wordmark.svg` | Горизонтальный логотип с текстом FK.market |

## Как сменить логотип на всём сайте

Выбранная концепция используется везде через файл `assets/logo-a.svg`. Чтобы сменить:

1. Откройте `logo.html` и выберите концепцию.
2. Скопируйте содержимое нужного файла (`logo-b.svg` / `logo-c.svg` / `logo-d.svg`) в `logo-a.svg`.
3. Логотип сменится во всей шапке, футере, favicon и бренд-буке.

## Технологии прототипа

- **Tailwind CSS** через CDN
- **Inter** (Google Fonts)
- Vanilla JS для темы и общего лэйаута (работает офлайн, без fetch)

При переходе в Next.js разметка переносится 1:1 в компоненты — Tailwind-классы одинаковые.

## Палитра бренда

| Роль | Цвет |
|------|------|
| Brand gradient | `#6366f1` → `#8b5cf6` → `#ec4899` (indigo → violet → pink) |
| Accent / success | `#10b981` (emerald) |
| Warning | `#eab308` (yellow) |
| Danger | `#ef4444` (red) |
| Bg light / surface | `#f9fafb` / `#ffffff` |
| Bg dark / surface | `#020617` (slate-950) / `#0f172a` (slate-900) |
